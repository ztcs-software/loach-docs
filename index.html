Loach

A native, local-first AI workspace for desktops. Run local LLMs with Ollama, or connect any OpenAI-compatible endpoint side-by-side — with a focused UX, native apps for Windows and Linux, and powerful features out of the box.

What is Loach?

Loach is an all-in-one desktop AI workspace built around a single idea: talking to an LLM should feel effortless. Simple from the first click and ready to grow with you as your needs do. It talks to a local Ollama server and accepts any OpenAI-compatible endpoint as a provider — including llama.cpp, LM Studio, vLLM and LiteLLM.

With local models — Qwen, Gemma, DeepSeek, GPT-OSS, Mistral and others — your data stays on your machine. There is no telemetry, no required network access, no paid subscriptions and no usage limits. Optional API keys live in your operating system's credential manager.

Behind a calm, beautifully crafted UI sits a rich feature set — ready when you need it, out of the way when you don't.

Highlights

Two providers, side-by-side
Switch between a local Ollama server and any OpenAI-compatible endpoint right from the chat header.
Spaces
Group chats around a project, with shared instructions, reference files and memory.
Snippets
Reusable prompts with an optional pinned model — click Run and a fresh chat is pre-filled.
Per-chat parameters
Tune temperature, top_k, top_p, min_p, context length and more — layered over Modelfile defaults.
MCP support
Register Model Context Protocol servers, inspect tools, and use them from chats.
Local-first & private
Everything lives in a local SQLite file. API keys go to the OS credential manager — never to disk.
